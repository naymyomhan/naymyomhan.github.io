## 🚀 Live Site

[https://naymyomhan.github.io](https://naymyomhan.github.io)

## 🛠️ Tech Stack

- Pure HTML, CSS, JavaScript (no frameworks)
- Custom animations, scroll effects & cinematic hero video
- Animated preloader, film-grain + vignette overlays, particle ocean
- Light / Dark pirate theme toggle (persisted)
- Mobile responsive design
- GitHub Pages hosting
